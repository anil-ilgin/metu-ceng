MDeePred - Web

This is a the test website for web implementation of 488 project.

How to use this service?

1) You need to choose the target protein family so we can make the predictions more accurate.

2) You need to choose which ID type you are using, so we can arrange the data according to that

3) You can use either CHEMBL ID of chemicals, or SMILES representations of them, but noth both. You need to choose which type are you using.

4) Making predictions can take time. Therefore, not to bore you down on this website, we ask your email so we can send the interaction values as csv file. We don't store or share your email anywhere.
